Parallel Portals

What is it?

Parallel Portals is a game I made for the Mini Jame Gam #10. I was tasked with making a game in 58 hours that conformed to the theme “Reality is breaking” and to include a phone in some way. It’s mainly an action platformer where you must find the 8 core crystals to mend torn apart universe. Throughout your adventure, you find portals into parallel realities with unique creatures and physics. Additionally, you can harness the power of realities other than your own by using your phone to create pockets of another universe. In the end, it ranked 6th place for the jam!

How?

This was my second game jam, and I had a lot more experience than the last time I did a jam. I was much better and managing my time and keeping the project limited to a reasonable scope. I initially intended for the game to be more linear, with levels that slowly introduced each universe and power, but I quickly realized that I only had time for one big level instead of several smaller ones.

This was made in GameMaker 1.4 and I took advantage of several pre-made asset packs for the visuals and sounds. The idea of multiple parallel universes would not have been possible without a set of symmetrical assets, so that was a big help.

After my initial submission, I went back and made a Version 2. This mostly consisted of bug fixing, performance improvements, adding a small tutorial to help guide players and adding a browser version. If you're interested, I wrote a in-depth dev-log about the development of this game on my game development website, which you can find here.

Status

This project is complete with no future updates planned. It is free to play from the itch.io page which you can access here.
