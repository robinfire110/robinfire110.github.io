Bindin'

What is it?

Bindin’ is a game I made for the GTMK Game Jam 2021 in collaboration with a friend. We had 48 hours to put together the game based on the theme, “Joined Together”. The idea we came up was a puzzle game where you have the ability to bind different control inputs (i.e. WASD) to various objects in order to interact with them and solve puzzles.

How?

The game was made in GameMaker Studio 1.4. I handled all the programming while my partner did the artwork. We only had 48 hours to put it together and this being our first game jam, there ended up being many things we didn’t get to implement. After the initial development period, I went back and made Version 2 which mostly comprised of bug fixes and adding the last few puzzles we didn’t have time to add. Just recently, I updated it to Version 2.1 which added a web version that can be played in the browser.

I wrote a in-depth dev-log about the development of this game on my game development website, which you can find here if you are interested.

Status

This project is complete with no future updates planned. It is free to play from the itch.io page which you can access here.
