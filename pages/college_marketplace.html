College Marketplace

What is it?

College Marketplace was a project made for the SEO Tech Developer program during the Summer of 2022. It was the final project for the program, and we were given a week to create some sort of useful product using the skills we learned. Our team decided to make an online marketplace for students. The idea was to create a site similar to Craigslist that was made specifically for college students. If you were a student who had something to sell, you could do so safely with other people on your campus or nearby campuses. Some of the features to support this goal was only allowing “.edu” emails to make an account, email verification, and a built-in messaging system so users don’t have to give away any emails or phone numbers. It has a lot of other neat features such as a user dashboard to edit listings, user reviews, and more.

How?

During the SEO Tech Developer program, we generally used Python for our projects, and this is no different. All the backend was done using Python and Flask. Our frontend was done with CSS, Jinja, and a lot of hard work from one of our team members, Emmanuel. We used SQL to store user data into a database and made sure to encrypt (by hashing and salting) important data such as passwords. We also used several APIs for certain features such as email verification and Google Maps integration (which allows users to get a rough estimate of how close they are to a seller).

Out of our group of five, Emmanuel and Nicole worked on the user interface and front-end while Angela, Sarah, and I took care of the backend. Some of my specific contributions include implementing the database for logins and item listings, incorporating login sessions, adding the Google Maps API and general project management. It was my first time really working with a bigger team, so there was a lot of growing pains as we figured it out. However, this experience has undoubtedly helped me grow as a team player and leader for future projects.

Status

Originally, the project was deployed to Heroku. However, in November 2022, Heroku changed how they operate and made it so we would have to pay to keep the application available. Sadly, it doesn’t make much sense to pay for a program purely for a portfolio. I tried to migrate it to another similar hosting site, but the amount of work needed to properly move it is also too much just to have on a portfolio. If you would like to check it out, you can still do so by downloading the source files on this page and following the instructions in the README. It’s not the most elegant solution and some things might not work, but it’s there if you want to try it out. Otherwise, I tried to take as many screenshots as I could to give you a good idea of how it works.
